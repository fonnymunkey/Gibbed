# Gibbed  
Originated as a fork of MobDismemberment (https://github.com/iChun/Mob-Dismemberment) by iChun.  
LGPLv3 maintainted from the original however has been essentially fully rewritten.  
  
Bursts entities into dynamically generated gib models based on the dying entity's model, including custom rendered models like Lycanite OBJ models.
